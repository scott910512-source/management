# ManagePilot — 기능 및 구성 정리

생산 통합관리 모듈 (TotalPilot 허브 하위).
회사 보안(IRM) Daily Report 엑셀을 자동으로 평문화하여,
공장별 생산현황·재고·수율을 집계·시각화한다.

---

## 1. 전체 구조

- 로그인 → `/hub` (TotalPilot 허브)
  - ManagePilot (`/production`) — 본 문서
  - StockPilot (`/`) — 재고관리(별도)

**모듈 전환**
- 사이드바 로고 클릭 → ManagePilot ↔ StockPilot 즉시 전환
- 상단 홈 버튼 → 허브로 이동

**멀티 공장**
- 데이터는 `1공장` / `2공장` 폴더로 분리 저장
- 총괄관리자(plantScope=all)는 공장 탭 · 통합뷰 사용

---

## 2. 데이터 파이프라인

보안 엑셀은 서버가 직접 못 읽으므로, 아래 순서로 평문화한다.

1. 보안 xlsx (IRM 암호화)
2. 사용자 PC의 Excel(COM)로 열어 셀 값 추출
3. 평문 CSV 생성
   - `daily-latest.csv` — 최신 날짜 시트
   - `batch-yield.csv` — 배치별(원료 사용량) 시트
   - `daily-<연>-<월>.csv` — 월말 스냅샷
4. 서버가 공장별 셀매핑으로 파싱
5. 종합현황 대시보드 표시

**추출 스크립트 (scripts/)**

- `manage-export.ps1` — 추출 로직 (ASCII 전용)
- `manage-config.txt` — 공장별 경로 설정 (UTF-8)
- `run-export.bat` — 실행 런처

**특징**

- CSV는 보안 라벨을 못 가져 항상 평문
- 복사 금지 우회: 시트 복사 대신 셀 값만 직접 읽음
- 파일명 자동 생성: `{연}`/`{월}` 치환 (월 바뀌어도 자동)
- 폴더 목록조회 차단 대응: 파일명 직접 생성해 바로 열기
- 시트 자동 인식: 최신 날짜시트 + "원료 사용량" 배치시트

---

## 3. 종합현황 대시보드

**최상단**

- 롤링 경고 배너: 수율 미달·재고 부족 순환 표시
- 상태바: 기준일 · CSV 갱신시각 · 기준월 선택
- 보고서 / 통합 보고서 / 수동 갱신 버튼
- 공장 탭 (총괄관리자)

**요약 카드 (4종)**

- 당월 생산 실적 (kg)
- 평균 생산 달성율
- 수율 주의 품목 (목표대비 -N.N%p)
- 재고관리 필요 (안전재고 이탈)

**품목별 계획 달성 현황 (표)**

- 컬럼: 일생산 · 계획 · 실적 · 달성율 · 연 달성율 · Batch · 수율(월/년)
- 표시 컬럼은 관리자가 선택
- 수율 셀: `74.2% (-2.2%p)` · 클릭 시 목표/미달 표시
- 품목 옆 배치수 배지

**재고 현황 (can)**

- 충진 · 출하 · 잔여 · 잔여(개월)
- 잔여 개월수 = 잔여 ÷ 월 소비량
- 색상: 안전재고 기준 (미만 빨강 / 1.5배 미만 주황 / 이상 초록)

**월별 추이 (선그래프, 클릭 시 품목×월 표)**

- 월별 생산량(kg): 품목별 선 + 총생산량 선
- 월별 수율(%): 품목별 선
- Y축 = 총합 ×1.1 (잘림 방지)
- 0·음수·오류값은 공백 처리, 미래월은 "데이터 없음"

## 4. 통합뷰 (`/production/compare`)

- 총괄관리자 전용
- 1·2공장 좌우 비교 + 통합 요약
- 공장별 품목 요약 표 (실적/달성율/수율)

---

## 5. 관리자 기능

**관리자 설정 (`/production/settings`)**

- 파일 경로: 공장별 감시 폴더 + 경로 테스트
- 재고 기준정보: 연간계획(kg) · 포장단위(kg) · 안전재고(개월)
  - 월 소비량(can) = 연간계획 ÷ 12 ÷ 포장단위
- 표시 컬럼: 계획달성 표 컬럼 선택

**셀 매핑 (`/production/cellmap`)**

- 공장별 엑셀 열 문자(B·K·AF …) 지정
- 행은 제품명으로 자동 탐색
- 재고 개별 셀(절대 위치) 오버라이드
- 배치시트 매핑: 품목별 Batch no / 생산량 / Yield 열
- 미리보기: 인식 품목·추이·불일치 진단

**경고 이력 (`/production/alerts`)**

- 수율 미달·재고 부족을 일자별 1회 자동 기록
- 유형 필터(전체/수율미달/재고부족) 조회

---

## 6. 보고서 내보내기 (HTML)

- 보고서: 현재 공장 / 통합 보고서: 1+2공장
- 구성: 요약 · 계획달성 · 재고 · 월별 생산/수율(차트+표) · 경고
- 애플풍 카드, 인쇄(PDF) 최적화

---

## 7. 서버 API (`/api/production`)

- `GET /data` — 종합현황 (파싱·재고·경고 산출, 이력 기록)
- `GET /data?period=2026-6` — 6월 말 스냅샷
- `POST /test-path` — CSV 인식 미리보기(진단)
- `GET /alerts` — 경고 이력 (관리자)

**데이터 저장 (CSV)**

- `settings` (공장별): 경로·셀매핑·재고기준·표시컬럼
- `prod_alerts` (전역): 경고 이력 (최근 3000건)
- `login_logs` (전역): 로그인 이력

---

## 8. 운영 체크포인트

- 추출 스크립트가 돌아야 CSV 최신화 (스케줄러 권장)
- 추출 PC는 로그온·전원 ON 상태여야 복호화 가능
- 재고 경고: 기준정보(연간계획·포장단위·안전재고) 입력 필수
- 수율 경고: 원본에 목표 수율 존재 + 셀매핑 일치 필요
- 월별 추이: 배치시트 매핑까지 넣어야 채워짐
- 품목명: 종합현황과 배치시트 철자 동일해야 연결됨

---

## 9. 용어

- daily-latest.csv: 최신 날짜 시트 (일별 현황)
- batch-yield.csv: 배치별 시트 (월별 추이 원천)
- 셀 매핑: 엑셀 열/셀 위치를 공장별로 지정
- 잔여 개월수: 현재 재고로 버티는 개월 (잔여 ÷ 월소비)
- 안전재고(개월): 이 값 미만이면 재고 부족 경고
