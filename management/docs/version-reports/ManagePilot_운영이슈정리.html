1

구축 중 겪은 문제점

보안 엑셀을 서버가 자동으로 읽게 만드는 과정에서 발생한 이슈와 해결 방식.

보안(IRM) 암호화로 서버가 엑셀을 못 읽음 해결
원인
정보보호 라벨이 걸린 파일은 서버가 복호화 권한이 없음.
해결
권한 계정 PC의 Excel로 열어 평문 CSV로 추출 → 서버는 CSV만 읽음. CSV는 라벨을 가질 수 없어 항상 평문.
“복사 금지” 권한으로 시트 내보내기 차단 해결
원인
권한이 “보기 가능, 복사 금지”라 시트 복사가 막힘.
해결
시트를 복사하지 않고 셀 값만 직접 읽어 CSV로 기록.
한글 인코딩 깨짐 (스크립트·배치파일) 해결
원인
스크립트 안의 한글이 다른 인코딩으로 읽히면 깨짐.
해결
실행 스크립트는 한글 0%(ASCII)로, 한글 경로는 별도 설정 텍스트파일(UTF-8)로 분리해 강제 UTF-8로 읽음.
공유폴더 목록 조회 차단 해결
원인
해당 공유는 파일 열기는 되지만 폴더 목록 조회는 거부.
해결
폴더를 뒤지지 않고 현재 월로 파일명을 만들어 바로 열기({연}·{월} 자동 치환).
월/공장별 파일·양식 차이 해결
원인
월이 바뀌면 파일명이 달라지고, 공장마다 열 위치·시트명이 다름.
해결
파일명 자동 생성 + 공장별 셀 매핑(열 위치를 관리자 설정에서 지정). 최신 날짜시트/배치시트 자동 인식.
원본 파일 잠김·열기 암호 외부 요인
원인
다른 사용자가 편집 중이거나, 열기 암호가 걸린 파일.
완화
스크립트가 전월 파일 자동 시도 + 어떤 파일을 열었는지 로그. 근본 해결은 파일 닫기/암호 제거(운영 측).
2

운영하면서 놓치면 안 되는 부분

아래 설정이 비어 있으면 해당 지표·경고가 표시되지 않는다. 최초 1회 세팅 필수.

📉 수율 주의 품목 (경고) 설정 전제
동작
품목별 실적 수율 < 목표 수율이면 상단 요약카드·롤링배너·경고이력에 자동 표기. 목표 대비 -N.N%p.
전제
원본에 목표 수율이 있어야 함(수율 행). 셀 매핑에서 수율 열이 맞아야 판정됨.
기준
5%p 이상 미달 = 빨강(중요), 그 미만 = 주황.
📦 재고관리 필요 (경고) 기준정보 필수
동작
품목별 잔여 개월수 < 안전재고(개월)이면 “재고관리 필요” 경고. 잔여(개월) 색상도 이 기준으로 갈림.
전제
관리자 설정 > 재고 기준정보에 연간계획(kg)·포장단위(kg)·안전재고(개월) 입력.
월 소비량(can) = 연간계획 ÷ 12 ÷ 포장단위. 미입력 시 잔여개월 계산 불가 → 경고 안 뜸.
🧭 셀 매핑 (양식 연동) 공장별 필수
필요
공장마다 열 위치가 다르므로 공장별로 열 지정 필요. 안 맞으면 숫자가 비거나 엉뚱하게 나옴.
배치
월별 추이·배치수는 배치시트 매핑(품목별 Batch no/생산량/Yield 열)까지 넣어야 채워짐.
확인
미리보기로 인식 품목·추이·불일치를 확인하고 저장.
🔁 데이터 갱신 흐름 운영 리듬
  • 추출 스크립트가 실행돼야 CSV가 최신화됨 (수동 실행 또는 스케줄러).
  • 화면은 CSV를 읽음 → 스크립트 안 돌면 화면도 그대로(어제 값).
  • 추출 PC가 로그온·전원 ON 상태여야 보안 복호화가 됨(자동 실행 조건).
  • 화면 새로고침/수동갱신/자동(1시간)은 그 시점 CSV를 다시 읽음.
🗓️ 기준월(월말 스냅샷) · 경고 이력 참고
스냅샷
월말 시점 화면을 연-월별로 아카이브. 원본 시트를 지워도 이미 찍은 스냅샷은 유지. 상단 기준월에서 선택.
이력
수율 미달·재고 부족은 (공장·기준일·품목)당 하루 1회 자동 기록 → 경고 이력 조회. 단, 화면을 한 번은 열어야 그날 기록됨.
3

데이터 연동 시 주의사항

원본 양식·환경이 바뀌면 아래를 점검한다.

  • 시트 이름 규칙: 날짜시트(예: N월 N일), 배치시트(원료 사용량) 이름이 유지돼야 자동 인식됨. 바뀌면 인식 실패.
  • 열 위치 변경: 원본에서 열이 추가/이동되면 셀 매핑을 다시 맞춰야 함.
  • 품목명 일치: 종합현황 품목명과 배치시트 품목명 철자가 같아야 월별 추이가 연결됨.
  • 단위 일관성: 수율은 비율(0.78) 기준으로 처리(×100). 원본이 % 숫자로 바뀌면 조정 필요.
  • 오류·극소값: 음수·오류값·10% 미만은 0으로 간주해 그래프에서 공백 처리(왜곡 방지).
  • 월 전환: 파일명은 자동으로 현재 월을 사용. 파일 저장 방식(폴더/이름)이 바뀌면 설정 1회 갱신.
  • 파일 잠김: 원본을 편집 중이면 추출 실패 가능 → 추출 시간대를 편집과 겹치지 않게.
