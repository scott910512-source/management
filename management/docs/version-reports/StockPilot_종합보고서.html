1. 전체 개발 흐름

수기관리 통합(v1)에서 출발해 재고 추적 → 운영 자동화 → 멀티공장·권한 → AI·고도화로 단계적으로 발전했습니다.

v1초기 구축
종이·엑셀 수기관리를 React+Express+CSV 한 시스템으로 통합. 로그인·가입승인 기반 마련.
v2Lot 기반 재고
입고일별 Lot 단위 추적, 품목 기준정보(단위·안전재고·업체) 체계화.
v3운영 대시보드
경고·퀵메뉴·Task를 한 화면에. 선입선출(FIFO) 위반 자동 감지 시작.
v3.1UI/UX 개선
제품별 그룹핑·색상·뱃지로 한눈에 읽히는 현황 화면으로 정리.
v4멀티 공장
1·2공장 데이터 완전 분리, 서버 시작 시 자동 백업.
v5역할 5종
관리자·공장관리자·팀관리자(조회전용)·사용자로 권한 분리, 공장 간 격리.
v6AI 검색·메뉴얼
폐쇄망 호환 자연어 검색 엔진, 사용자 메뉴얼 추가.
v7현행 고도화
다량 입고·FIFO 분배, 합성 Batch 투입이력, 유해물질 관리대장, 월간 보고서.

2. 버전별 포커스 — 흐름 · 인터락 · 편의

각 버전이 무엇을 풀었는지, 어떤 안전장치(인터락)와 편의기능을 더했는지 정리했습니다.

v1초기 구축
흐름수기 → 시스템 — 흩어진 종이·엑셀 기록을 한 곳에서 입력·조회
인터락가입신청 → 관리자 승인 — 승인 전 접근 차단(무단 가입 방지)
편의CSV 내보내기 — 엑셀에서 바로 여는 다운로드(BOM으로 한글 안 깨짐)
v2Lot 기반 재고
흐름입고 → Lot 추적 → 출고 차감 — 어느 Lot이 얼마 남았는지 실시간 반영
인터락이중 안전재고 — % 기준 + 기준수량 동시 설정으로 부족 누락 방지
편의품목 기준정보 자동화 — 단위·업체·제품을 미리 등록해 입력 단순화
v3운영 대시보드
흐름한 화면 운영 — 경고·퀵메뉴·진행 Task를 종합현황에서 동시에
인터락FIFO(선입선출) 인터락 — 오래된 Lot 두고 새 Lot 쓰면 경고, 강제 시 이상발생 자동기록
편의퀵메뉴 — 입고/사용/수불을 첫 화면에서 바로 시작
v3.1UI/UX 개선
흐름읽히는 현황 — 제품별 그룹핑 + 상태 뱃지로 흐름이 한눈에
인터락부족 빨간색 강조 — 안전재고 미달 행을 강제 노출해 놓치지 않게
편의품목 선택 자동 채움 — 단위·업체·Lot 양식이 자동 입력 → 오입력 감소
v4멀티 공장
흐름공장별 분리 운영 — 1·2공장 데이터를 완전히 나눠 혼선 차단
인터락데이터 보존·자동 백업 — 업데이트가 자료를 덮어쓰지 않고, 켤 때마다 백업
편의공장 선택기 — 권한 있는 공장만 드롭다운으로 즉시 전환
v5역할 5종
흐름역할별 화면 — 권한에 맞는 메뉴·버튼만 노출
인터락조회전용 전역 차단 — 팀관리자의 모든 쓰기 요청을 서버에서 403 차단
인터락공장 경계 강제 — 타 공장 데이터 접근 시 거부(403)
v6AI 검색·메뉴얼
흐름자연어로 묻기 — "이번달 헥산 사용량" 같은 말로 바로 조회
인터락숫자 비조작 — 실데이터로만 계산, 폐쇄망에서 외부 LLM 없이 동작
편의예시 칩 + 메뉴얼 — 자주 쓰는 검색어 바로가기, 이미지 안내 페이지
v7현행 고도화
편의일괄 Lot 입고 — 같은 품목 Lot을 번호 범위(A01~A20)로 한 번에 등록
편의잔량 자동 계산·[전량] 버튼 — 사용/입고 후 잔여 자동 표시
인터락FIFO 자동 분배 — 일괄 출고 시 가장 오래된 Lot부터 자동 차감
인터락Task 완료 승인 · 유해물질 보관한도 경고 — 임의 완료 방지, 위험 보관 차단
흐름합성 Batch 투입이력 · 월간 보고서 — Batch별 투입 추적, 관리자 보고서 자동 생성

3. 인터락(안전장치) 총정리

실수·사고·무단 접근을 시스템이 막아주는 장치들입니다.

가입 승인 절차
승인 전 접근 차단으로 무단 가입·외부 노출 방지
FIFO 선입선출 인터락
오래된 Lot 우선 사용 강제, 위반 시 경고+이상발생 자동기록
조회전용 전역 차단
팀관리자 쓰기 요청을 서버에서 일괄 403
공장 경계 격리
타 공장 데이터 접근 거부, 업데이트 시 자료 보존
Task 완료 승인
일반 사용자 완료는 대기 → 관리자 승인 시 확정
유해물질 보관한도 경고
설정 % 초과 시 경고로 위험 보관 차단
Canister 90% 경고
사이즈별 최대 무게 임박 시 빨간 테두리 경고
CSV BOM·동시쓰기 잠금
한글 깨짐·파일 충돌 방지

입력 수량을 오래된 Lot부터 자동으로 나눠 차감
잔량 자동 계산·[전량]
사용/입고 후 잔여수량 자동 표시, 전량 버튼
품목 선택 자동 채움
단위·업체·Lot 양식 자동 입력
퀵메뉴

AI 자연어 검색
말로 묻고 표·답변으로 즉시 조회
Batch 번호 자동·연도 리셋
품목별 다음 번호 자동 채움
