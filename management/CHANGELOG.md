## v4 · 멀티 공장 (1공장 · 2공장 분리)

| 구분 | 내용 |
|---|---|
| 신규 | 공장별 완전 데이터 분리 (`server/data/1공장/` · `2공장/`) |
| 신규 | 사이드바 공장 선택기 (드롭다운, 권한 있는 공장만 표시) |
| 신규 | X-Plant 헤더로 공장 컨텍스트 전달 |
| 신규 | 자동 백업 — 서버 시작 시 `server/backups/` 에 스냅샷 생성 |
| 신규 | 최근 20개 백업만 유지, 오래된 것 자동 삭제 |
| 신규 | 기존 데이터 마이그레이션 — 루트 CSV → 2공장으로 자동 이동 |
| 신규 | 업데이트 시 `data` 폴더 절대 덮어쓰지 않음 |
| 버그 수정 | X-Plant 한글 헤더 오류 → `encodeURIComponent` / `decodeURIComponent` 처리 |
| 버그 수정 | `mutate('users', null, fn)` — 글로벌 테이블은 plant=null 명시 |
| 보안 강화 | `isSuper` 조건 — `plantScope==='all'` 명시적으로만 허용, 빈 값 차단 |

| v6 AI 검색 배치 | 종합현황 **상단 전체폭** | 🔄 변경 | **퀵메뉴(좌) + AI검색(우)** 한 줄 배치 |
