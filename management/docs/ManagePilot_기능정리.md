# ManagePilot — 기능 및 구성 정리

> 생산 통합관리 모듈 (TotalPilot 허브 하위). 회사 보안(IRM) Daily Report 엑셀을
> 자동으로 평문화하여 공장별 생산현황·재고·수율을 집계·시각화한다.

---

## 1. 전체 구조

```
로그인 → /hub (TotalPilot)
          ├─ ManagePilot (/production)   ← 본 문서
          └─ StockPilot  (/)             ← 재고관리(별도)
```

- **모듈 전환**: 사이드바 로고 클릭으로 ManagePilot ↔ StockPilot 즉시 전환, 상단 🏠 홈 버튼으로 허브 이동
- **멀티 공장**: 데이터는 `server/data/1공장`, `server/data/2공장` 폴더로 분리 저장
- **권한**: 총괄관리자(`plantScope=all`)는 공장 탭/통합뷰 사용, 관리자는 설정·경고이력 접근

---

## 2. 데이터 파이프라인 (보안 엑셀 → 화면)

```
보안 xlsx(IRM)  →  [사용자 PC] Excel COM 셀값 읽기(PowerShell)  →  평문 CSV
                                                                   ├─ daily-latest.csv (최신 날짜 시트)
                                                                   └─ batch-yield.csv  (배치별 수율 시트)
        →  [서버] 공장별 셀매핑으로 파싱  →  종합현황 대시보드
```

### 추출 스크립트 (`scripts/`)
| 파일 | 역할 |
|---|---|
| `export-daily-report-csv.ps1` | 보안 엑셀의 최신 날짜시트·배치시트를 UTF-8 CSV로 추출 (ASCII 전용 소스, 인코딩 안전) |
| `run-daily-export.bat` | 1·2공장 일괄 추출 (작업 스케줄러 등록용) |

- **원리**: CSV는 구조상 보안 라벨을 못 가지므로 항상 평문. "복사 금지" 권한 우회를 위해 시트 복사 대신 **셀 값을 직접 읽음**
- **자동 선택**: `N월 N일` 중 최신 → `daily-latest.csv`, `배치별…수율` → `batch-yield.csv` (날짜/연도 바뀌어도 자동 인식)
- **운영**: 작업 스케줄러로 매일 자동 실행 → 사용자는 화면만 열면 최신 데이터

## 3. 종합현황 대시보드 (`/production`)

### 3.1 최상단
- **롤링 경고 배너**: 수율 미달(±%p)·재고 부족을 실시간 집계해 순환 표시
- **상태바**: `기준일`(CSV 보고일) · CSV 갱신시각 · 📄 보고서 · 📑 통합 보고서 · 🔄 수동 갱신
- **공장 탭**(총괄관리자): 1공장 / 2공장 전환

### 3.2 요약 카드 (공장별 4종)
| 카드 | 내용 |
|---|---|
| 당월 생산 실적 | 총 실적(kg) |
| 평균 생산 달성율 | 품목 계획 대비 평균 |
| 수율 주의 품목 | 목표 대비 가장 미달 품목 (`-N.N%p`) |
| 재고관리 필요 | 안전재고 이탈 품목 (잔여 개월수) |

### 3.3 품목별 계획 달성 현황 (표)
- 컬럼: 품목(색 동그라미+배치수 배지) · 6월계획 · 6월실적(전월대비 ▲▼%) · 6월 달성율(달성/진행/미달 배지) · 연 생산 달성율 · 수율(월) · 수율(년)
- **표시 컬럼은 관리자 설정에서 선택**
- 수율 셀: `74.2% (-2.2%p)` 표기, 클릭 시 `목표 78.7% 미달` 노출

### 3.4 재고 현황 (can)
- 충진 · 출하 · 잔여 · **잔여(개월)**
- 잔여 개월수 = 잔여수량 ÷ 월소비량
- **색상 = 안전재고(개월) 기준**: 미만=🔴, ×1.5 미만=🟠, 이상=🟢

### 3.5 월별 추이 (선그래프, 클릭 시 품목×월 내역표)
- **월별 생산량(kg)**: 품목별 선 + 총 생산량 선, Y축 = 총합×1.1
- **월별 수율(%)**: 품목별 수율 선
- 10% 미만·음수·오류값은 공백 처리, 미래월은 "데이터 없음", 끝값 라벨 자동 겹침 회피

## 4. 통합뷰 (`/production/compare`, 총괄관리자)
- 1·2공장을 좌우로 나란히 비교
- 상단 통합 요약(당월 실적 합계 · 대상 공장 · 경고 합계)
- 공장별 요약 카드 + 품목별(실적/달성율/수율) 미니 표

---

## 5. 관리자 기능

### 5.1 관리자 설정 (`/production/settings`)
| 섹션 | 내용 |
|---|---|
| Daily Report 파일 경로 | 공장별 감시 폴더 경로 지정 + 경로 테스트 |
| 재고 기준정보 | 품목별 연간계획(kg)·포장단위(kg)·안전재고(개월). 월소비량(can) = 연간계획 ÷ 12 ÷ 포장단위 (자동, 수정가능) |
| 계획 달성 현황 표시 컬럼 | 표에 노출할 컬럼 선택 |

### 5.2 셀 매핑 (`/production/cellmap`)
- 공장별로 엑셀 **열 문자(B·K·AF…)** 지정 → 양식이 달라도 코드 수정 불필요
- 행은 제품명 자동 탐색
- **재고 개별 셀(절대 위치)**: 특수 양식(예: BTBAS 합계 잔여 `AN10`) 오버라이드
- **배치시트 매핑**: 월 라벨 열·소계 라벨·품목별 Batch no/생산량/Yield 열
- **미리보기**: 인식 품목·배치 추이·불일치 진단 노출

### 5.3 경고 이력 (`/production/alerts`)
- 수율 미달·재고 부족을 **(공장·기준일·품목·유형)당 하루 1회** 자동 기록
- 유형 필터(전체/수율미달/재고부족) 조회

## 6. 보고서 내보내기 (HTML 다운로드)
- **📄 보고서**: 현재 공장 / **📑 통합 보고서**: 1+2공장
- 구성: 요약 · 품목별 계획달성(일생산 제외) · 재고현황 · 월별 생산량/수율(선그래프 SVG + 표) · 경고사항
- 애플풍 카드, 인쇄(Ctrl+P → PDF) 최적화

## 7. 서버 API (`/api/production`)
| 메서드 | 경로 | 설명 |
|---|---|---|
| GET | `/data?plant=` | 종합현황 데이터(daily+batch 파싱, 재고·경고 산출). 경고 이력 자동 기록 |
| POST | `/test-path` | 폴더의 daily/batch CSV 인식 미리보기(+진단) |
| GET | `/alerts` | 경고 이력 조회(관리자) |

### 데이터 저장 (CSV)
| 테이블 | 위치 | 비고 |
|---|---|---|
| `settings` | 공장별 | 파일경로·셀매핑·재고기준·표시컬럼(JSON) |
| `prod_alerts` | 전역 | 경고 이력 (최근 3000건 유지) |
| `login_logs` | 전역 | 로그인 이력 |

## 9. 용어
- **daily-latest.csv**: 최신 날짜 시트(일일 생산·재고 현황)
- **batch-yield.csv**: 배치별 생산량·수율 시트(월별 추이 원천)
- **셀 매핑**: 엑셀 열/셀 위치를 공장별로 지정하는 설정
- **잔여 개월수**: 현재 재고로 버틸 수 있는 개월 (잔여 ÷ 월소비)
- **안전재고(개월)**: 이 값 미만이면 재고 부족 경고
